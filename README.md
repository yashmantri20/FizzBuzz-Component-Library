# FizzBuzz React Component Library

FizzBuzz UI provides a set of accessible, reusable, and composable React components that make it super easy to create websites and apps.

## Components 

0. Alert
1. Button
2. Avatar
3. Scroll To Top
4. Tooltip
5. Image
6. Spinner
7. Progress Bar

## Installation

```
npm i fizzbuzz-ui
OR
yarn add fizzbuzz-ui
```
